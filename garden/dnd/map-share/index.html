CARTOMANCER

Live map control for the table

A self-hostable D&D map display system. The DM controls everything from a tablet — the table screen follows in real time. No apps. No accounts. Just a browser, a map, and a session key.

DM Portal Source

How it works

01

DM logs in

Enter your account key on the DM portal. Upload your maps. Create a session and get a 6-character session ID.

02

Table connects

Open the table URL on the display device. Type the session ID. The table is now slaved to the DM portal.

03

Play

Select a map. Place tokens. Drop a movement ring. Adjust color grade for the TV. Everything updates on the table in real time.

⬡

Grid calibration

Tap two corners of a cell on the map preview. The grid snaps to the physical map instantly.

◎

Movement rings

Right-tap any point to project an animated radius ring showing movement range.

◑

Day / night

Simulate time of day with real-time color overlays. Dawn, dusk, and night — no second asset needed.

◈

Color grading

Live contrast, saturation, and brightness sliders. Compensate for any TV colour profile on the fly.

⟳

Auto-reconnect

Both devices reconnect automatically. Session state is persisted to disk on every change.

⊞

Self-hostable

One Go binary, one Docker container, flat JSON files. Clone, configure, run.

Self-host in 5 minutes

git clone https://github.com/Tristan-Phillips/cartomancer
cd cartomancer && cp config.example.json data/config.json
# edit data/config.json — set admin_key
docker compose up -d

Runs behind a Cloudflare Tunnel or reverse proxy. No public ports. All data on your machine.
