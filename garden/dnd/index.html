Grimoire

Live map control for the table

A self-hostable D&D map display system. The DM controls everything from a tablet — the table screen follows in real time.

The Ritual
  1. I

    The DM enters

    Enter your account key on the DM portal. Upload your maps. Create a session and receive a six-character session code.

  2. II

    The table awakens

    Open the table URL on your display screen. Enter the session code. The table is now bound to the DM portal in real time.

  3. III

    Begin

    Select a map. Pinch, zoom, rotate. Drop the curtain between scenes. Shift the time of day. Every gesture on the DM's tablet echoes to the table instantly.

Capabilities
⬡

Grid Overlay

Toggle a calibrated grid on the live map. Control cell size and opacity with sliders that vanish after use.

⟳

Pinch & Rotate

Two-finger pinch-zoom and rotation on iPad. The table screen mirrors every gesture instantly over WebRTC.

◑

Day / Night

Atmospheric tint overlays simulate dawn, dusk, and night. No second asset required — just mood.

◎

Curtain Mode

A dramatic veil hides the map while you prepare the next scene. Lifts with a single tap.

✦

Annotations

Draw on the map with Apple Pencil or any stylus. Strokes sync live to the table. Two-finger tap to undo.

⊞

Self-Hostable

One Go binary, one Docker container, flat JSON files. Clone, configure, run. Everything stays on your machine.

Summon it yourself
shell
git clone https://github.com/Tristan-Phillips/grimoire cd grimoire && cp config.example.json data/config.json # edit data/config.json — set your admin_key docker compose up -d

Runs behind a Cloudflare Tunnel or reverse proxy. No public ports required. All maps and session data stay on your machine.
