GRIMOIRE

Live map control for the table

A self-hostable D&D map display system. The DM controls everything from a tablet — the table screen follows in real time. No apps. No accounts. Just a browser, a map, and a session key.

DM Portal Source

How it works

01

DM logs in

Enter your account key on the DM portal. Upload your maps. Create a session and get a 6-character session ID.

02

Table connects

Open the table URL on the display device. Type the session ID. The table is now slaved to the DM portal.

03

Play

Select a map. Rotate, zoom, pinch. Drop a curtain when you need a moment. Adjust time of day. Everything updates on the table in real time.

⬡

Grid overlay

Toggle a calibrated grid overlay on the live map. Slider controls cell size and auto-hides after 3 seconds.

⟳

Pinch & rotate

Two-finger rotate and pinch-zoom on iPad. The map orientation syncs to the table screen instantly.

◑

Day / night

Simulate time of day with atmospheric light and tint overlays. Dawn, dusk, and night — no second asset needed.

◎

Curtain mode

Hide the map from players while you prepare the next scene. A dramatic veil covers the table screen.

⎕

Multi-session

Run multiple sessions with password protection. Session state persists to disk on every change.

⊞

Self-hostable

One Go binary, one Docker container, flat JSON files. Clone, configure, run.

Self-host in 5 minutes

git clone https://github.com/Tristan-Phillips/grimoire
cd grimoire && cp config.example.json data/config.json
# edit data/config.json — set admin_key
docker compose up -d

Runs behind a Cloudflare Tunnel or reverse proxy. No public ports. All data on your machine.
